Open source Granny2 IO library written in C
------------------------------------------------------------

Operation step


1. Low level Granny2 library (KV parser, base compressor and decompressor)
2. High level Granny2 serializer (ArtTool, etc etc based from KeyNames)

Current support:
	Granny 2.1.0.3 nodes (FF6, no compression)
	
TODO support:
